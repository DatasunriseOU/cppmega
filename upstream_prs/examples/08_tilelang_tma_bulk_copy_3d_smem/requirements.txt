#       → LOG(WARNING) + graceful fallback to LowerNormalCopy (cp.async).

# (reference_stack_bench.md in MEMORY.md).
#

# CUDA should work for the reproducer, but to match the observed env keep 2.12.
#
# NOTE: TileLang is distributed as a source checkout on the bench machines
# (TL_TEMPLATE_PATH→/mnt/data/tilelang-build). pip install works for cold repro
# on clean boxes.
